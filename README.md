# Non-MarkovianAgent
In this project I simulate a daily trading scenario where a learning-based investor makes decisions based on the Sato-Crutchfield learning algorithm following non-Markovian rules. The learning algorithm is described in detail in the following reference:
Nicole, R., Sollich, P. & Galla, T. Stochastic evolution in populations of ideas. Sci Rep 7, 40580 (2017). https://doi.org/10.1038/srep40580
Data files included in this repository contain synthetic stock price data generated for demonstration purposes. Real historical data used in the project cannot be redistributed due to licensing restrictions.
